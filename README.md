The Quiet Cup Café 

A responsive café website built with HTML, CSS, and JavaScript

## Pages & Sections
- **Home** – Welcome message and opening hours
- **About** – Café story and values
- **Menu** – Coffee menu with images and prices
- **Reviews** – Customer testimonials
- **Contact** – Contact form and café details

## Built With
- HTML5
- CSS3
- JavaScript (Vanilla)

## Features
- Smooth scroll navigation
- Interactive contact form
- Responsive layout
- Clean and minimal design
